# Agent Instructions

## 规则

- Category `name` must be formatted as: "<emoji><space><Chinese name>".
- Level-1 Chinese name must be 2 Chinese characters.
- Level-2 Chinese name must be 4 Chinese characters.
- 当用户询问消费归类时，基于当前 categories.yaml 给出最合适的分类建议。
- 若现有分类无法覆盖，可适当调整 categories.yaml（新增或拆分二级分类，仍需满足 2/4 字规则）。
